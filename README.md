## Дискретная случайная величина и симулятор случайного блуждания
Веб-приложение на Streamlit для работы с дискретными случайными величинами (ДСВ), выполнения математических операций, визуализации распределений и моделирования случайных блужданий.

## Возможности
Управление ДСВ: Создание распределения с произвольными значениями и вероятностями с автоматической проверкой суммы вероятностей (должна равняться 1). Вычисление основных числовых характеристик: математическое ожидание, дисперсия, среднеквадратическое отклонение, коэффициенты асимметрии (skewness) и эксцесса (kurtosis).

Операции над ДСВ: Умножение на константу, сложение двух ДСВ (свёртка распределений), умножение двух ДСВ.

Визуализации: Таблица закона распределения, stem plot (диаграмма-стебель) для отображения вероятностей, полигон распределения (ломаная линия), функция распределения (кумулятивная CDF).

Сериализация: Сохранение ДСВ в JSON-файл с возможностью скачивания, загрузка ДСВ из JSON-файла, сброс к примеру по умолчанию.

Симуляция случайного блуждания: Шаги блуждания подчиняются выбранному распределению (текущая ДСВ, загруженная из файла или вновь созданная). Контроль процесса с помощью кнопок Старт и Стоп. Отображение траектории в реальном времени (каждый шаг занимает 1 секунду). Расчёт и вывод распределения конечного положения после всех шагов.

## Технологии
Python, Streamlit, Matplotlib, NumPy, JSON.

## Установка и запуск
Установите зависимости: pip install streamlit matplotlib numpy. Запустите приложение командой: streamlit run prob_theory.py.
